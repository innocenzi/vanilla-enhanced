# Vanilla Enhanced

A lightweight utility addon for Burning Crusade Classic. It keeps the default UI familiar while adding a handful of practical quality-of-life improvements for questing, bags, merchants, and other interfaces.

> [!WARNING]
> I did not write a single line of code. Read the source at your own risks. I will not be responsible for your ophtalmologist bill.

## Features

- Highly configurable quest markers on the world map and minimap
- Backpack sorting, searching and locking
- Scrap selling
- Automatic quest tracking by location
- UI quality of life, like binding quest log to the map and automatically opening bags

Every single feature can be disabled, and most can be configured to your liking.
