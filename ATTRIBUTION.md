# Vanilla Enhanced Attribution

This addon is intended for private/local use.

Quest location and French locale data are generated from a local Questie install, currently observed as Questie v11.29.5. Questie source:
https://github.com/Questie/Questie

The `target-threat` module is a local replacement for Target's Threat by TwistedFaith. It recreates the small target-frame threat percentage behavior for private/local use.

Vendored map libraries are copied from the local Questie distribution:

- LibStub
- CallbackHandler-1.0
- HereBeDragons-2.0
- HereBeDragons-Pins-2.0

Before any public distribution, review the licenses and attribution requirements for Questie-derived data and all vendored libraries.
